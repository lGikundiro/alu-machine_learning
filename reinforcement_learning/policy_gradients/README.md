# Policy Gradients

This directory contains implementations of policy gradient methods for reinforcement learning.

## Tasks

### 0. Simple Policy function
Computes the policy (softmax action probabilities) given a state matrix and weight matrix.

**File:** `policy_gradient.py`

### 1. Compute the Monte-Carlo policy gradient
Computes the action and gradient using the Monte-Carlo policy gradient method.

**File:** `policy_gradient.py`

### 2. Implement the training
Full training loop using Monte-Carlo policy gradients on the CartPole environment.

**File:** `train.py`

### 3. Animate iteration
Updated training with optional rendering every 1000 episodes to visualize learning progress.

**File:** `train.py`
