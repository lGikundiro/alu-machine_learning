# Temporal Difference

This directory contains implementations of Temporal Difference learning algorithms.

## Tasks

### 0. Monte Carlo
Performs the Monte Carlo algorithm for value estimation using first-visit returns.

**File:** `0-monte_carlo.py`

### 1. TD(λ)
Performs the TD(lambda) algorithm with eligibility traces for value estimation.

**File:** `1-td_lambtha.py`

### 2. SARSA(λ)
Performs the SARSA(lambda) algorithm with eligibility traces for Q-value estimation using epsilon-greedy action selection.

**File:** `2-sarsa_lambtha.py`
